# Go Demo — 学习项目

## 项目背景

这是一个 Go 语言学习项目，用于系统性地从基础到进阶学习 Go。

## 用户背景

- 10 年前端开发经验
- 熟悉 Node.js、Python
- 不需要解释基础编程概念，直接聚焦 Go 的特性和与 JS/Python 的差异
- 讲解时可用 JS/Node/Python 类比

## 学习进度

- [x] 基础语法：变量、类型、slice（len/cap、共享内存、copy、append 扩容）
- [x] 函数：多返回值、命名返回值、variadic 函数
- [x] 结构体 struct
- [x] 指针
- [x] interface
- [x] goroutine + channel
- [x] defer / panic / recover
- [x] error 处理模式
- [ ] Go modules 与项目结构
- [ ] 标准库：net/http、encoding/json
- [ ] 泛型（Go 1.18+）
- [ ] context
- [ ] 反射与性能优化

## 教学原则

- 每次专注一个主题
- 提供可直接运行的代码示例（在 helloworld.go 中练习）
- 对比 JS/Python 讲解差异，而不是从零解释编程概念
